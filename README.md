## File Naming & Trait Parsing
- PNG names start with an index: `000_fire_double-tribal.png`
- Background tokens: `bg-red`, `bg_red`, `bg red`, `mg-*` also accepted
- Elements detected from filename tokens or parent folders (fire, water, earth, air, void, electric, light, shadow, etc.)
- Types: `mushroom`, `shroom`, `agaric`, etc. map to Type=Mushroom; cannabis/plant aliases map to Type=Cannabis/Plant
- Strains: tokens like `kush`, `psilocybe`, `sacred mexica` are captured as `Strain`
- Mushroom strains (e.g., Sacred Mexica, Golden Teacher, Wavy Caps, Shaggy Mane, Psilocyben Cubensis) force Type=Mushroom even if no type token is present
- Motifs: `tribal`, `swirl`, and `double tribal` are Motif values (so `double` no longer falls into Variant)
- Accessories: `halo`, `hat`, `eyeglasses`, `crown`, etc.
- Colors: common color tokens are normalized and grouped into `Sprite Color`; background colors stay under `Background`
- Leftover tokens become `Variant`
